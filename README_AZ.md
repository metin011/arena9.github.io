# ⚽ Arena 9 - Futbol Statistika Sistemi

Flask ilə hazırlanmış peşəkar futbol statistika idarəetmə sistemi.

## 🚀 Tez Başlama

### Lokal işə salma:

```bash
pip install -r requirements.txt
python app.py
```

Sayt açılacaq: http://localhost:5000

**İlk giriş:**
- İstifadəçi: `admin`
- Şifrə: `admin123`
